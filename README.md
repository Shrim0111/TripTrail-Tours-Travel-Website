# TripTrail – Tours & Travel Website

TripTrail is a responsive tours and travel website developed to provide users with an easy and engaging platform for exploring travel destinations and tour packages. The website includes essential features such as a homepage, tour packages, gallery, booking form, customer reviews, and a contact page, delivering a seamless user experience across devices.

The project focuses on creating a clean, responsive, and user-friendly interface while integrating dynamic functionality for handling bookings and user interactions.

### Features

* Responsive and modern UI
* Browse travel destinations and tour packages
* Online booking form
* Customer reviews section
* Image gallery
* Contact page for inquiries

### Tech Stack

* **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Development Tools:** XAMPP, Visual Studio Code
